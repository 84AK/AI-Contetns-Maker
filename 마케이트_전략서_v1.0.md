## 3. 핵심 고통점 (Pain Points)

### Pain Point #1: 시간 부족 — "콘텐츠 만들 시간이 없어요"
소상공인의 하루는 가게 운영만으로 꽉 차 있다. SNS 하나 올리려고 사진 찍고, 편집하고, 문구 쓰는 데 1~2시간이 걸리는 건 현실적으로 불가능하다. 주 1회 업로드도 버거운 상황.

**현실 수치**: 체계적 디지털 마케팅 실행 비율 30% 미만 → 70%는 하고 싶어도 못 하고 있음

### Pain Point #2: 전문 지식 부족 — "포토샵, 피그마는 배울 엄두가 안 나요"
디지털 전환 준비도 '지식' 영역이 2.53/5점으로 낮음. 기존 디자인 툴은 학습 장벽이 높고, AI 툴은 영어 중심이어서 진입 장벽이 큼.

### Pain Point #3: 비용 부담 — "마케팅 대행사는 너무 비싸요"
- 마케팅 대행사 월 계약: 50~300만 원 이상 (추정)
- 소상공인 평균 마케팅 예산: 월 30~100만 원
- 디자인 프리랜서 1건 의뢰: 5~30만 원 수준
- 결론: 전문가를 쓸 여유가 없음

### Pain Point #4: 한국 맥락 부재 — "외국 툴은 한국 정서를 몰라요"
캔바(Canva) 등 글로벌 툴은 한국어 지원이 미흡하고, 한국 소상공인 특유의 업종별 분위기(ex. 카페 감성, 치킨집 홍보, 마트 세일 현수막 등)에 맞는 템플릿이 부족하다.

### Pain Point #5: 일관성 없는 브랜딩 — "올릴 때마다 느낌이 달라요"
브랜드 가이드라인이나 컬러 팔레트 개념 없이 즉흥적으로 만드는 콘텐츠는 브랜드 이미지를 오히려 망칠 수 있다. 그러나 브랜딩을 배울 여유도 없다.

### Pain Point #6: 결과를 모름 — "SNS 올려도 뭐가 효과인지 모르겠어요"
어떤 콘텐츠가 반응이 좋은지, 어떻게 개선해야 하는지에 대한 피드백 루프가 없음.

## 5. 차별화 전략 (우리만의 강점)

### 차별화 포인트 1: "1분 완성" AI 자동 생성

> "업종 선택 → 오늘의 메뉴/상품 입력 → 자동으로 SNS 포스팅 완성"

- 사용자가 텍스트 몇 줄만 입력하면 이미지+카피+해시태그까지 자동 생성
- 기존 툴(캔바, 미리캔버스)은 "빈 캔버스에서 시작" → 마케이트는 "결과물에서 시작"
- 목표: 콘텐츠 1개 제작에 **5분 이내** (기존 대비 80% 시간 절감)

### 차별화 포인트 2: 한국 업종 DB 기반 초개인화

- 카페, 음식점, 미용실, 의류매장, 학원, 부동산 등 **50개+ 업종별 AI 학습 데이터**
- 업종별 마케팅 문구 톤앤매너 자동 적용 (예: 치킨집은 "바삭함" 강조, 카페는 "오늘 하루 쉬어가세요" 톤)
- 한국 절기/이벤트 자동 감지 (설날, 삼겹살 데이, 밸런타인데이, 블랙프라이데이 등)

### 차별화 포인트 3: 모바일 퍼스트 설계

- 소상공인은 PC보다 스마트폰 위주 → 모바일 앱 최우선 설계
- 가게에서 음식 사진 찍고 → 앱에서 3탭으로 → 인스타 바로 업로드
- 기존 웹 기반 경쟁사와 정면 차별화

### 차별화 포인트 4: 브랜드 키트 자동 설정

- 첫 가입 시 가게 이름·업종·분위기만 입력하면 AI가 **컬러 팔레트 + 폰트 + 로고 스타일** 자동 제안
- 이후 모든 콘텐츠에 자동으로 브랜드 통일성 적용 → 전문 브랜딩 에이전시 효과
- 경쟁사 대비: 캔바는 브랜드 키트 기능이 있으나 소상공인이 직접 설정해야 함

### 차별화 포인트 5: 정부 지원 연계 채널 활용

- 2025년 소상공인 디지털 전환 지원 사업 (최대 300만 원 지원, 중기부·과기정통부)
- B2G 채널: 소상공인진흥공단, 지역 상공회의소, 소진공 파트너십을 통한 무료/저가 공급
- 경쟁사 중 정부 지원 연계를 체계적으로 하는 곳 없음 → 선점 기회

## 6. 핵심 기능 우선순위 (MVP)

| 우선순위 | 기능 | 설명 |
|----------|------|------|
| P0 | **AI SNS 포스팅 생성** | 텍스트 입력 → 이미지+카피 자동 생성 (인스타, 네이버 블로그 포맷) |
| P0 | **업종 선택 온보딩** | 50개+ 업종 중 선택 → AI 커스터마이징 |
| P0 | **모바일 앱 (iOS/Android)** | 스마트폰 퍼스트 경험 |
| P1 | **브랜드 키트 자동 생성** | 가게 정보 입력 → 컬러/폰트 자동 제안 |

| P2 | **해시태그 자동 추천** | 업종+지역 기반 해시태그 생성 |

### Phase 2 — Growth (4~8개월): "확장"

| 우선순위 | 기능 | 설명 |
|----------|------|------|
| P1 | **SNS 직접 발행** | 인스타그램·네이버 블로그 연동 예약 발행 |
| P1 | **AI 상품 사진 보정** | 폰 사진 → 전문 제품 사진으로 AI 업스케일 |
| P2 | **콘텐츠 성과 분석** | 조회수·좋아요·방문자 트래킹 |
| P2 | **팀 협업** | 직원이 콘텐츠 초안 올리면 사장이 승인 |

### Phase 3 — Scale (9개월~): "생태계"

| 기능 | 설명 |
|------|------|
| AI 광고 소재 생성 | 네이버 스마트플레이스·인스타 광고 소재 자동 제작 |
| 경쟁 가게 벤치마킹 | 주변 경쟁사 SNS 분석 |
| 마케팅 코치 AI | 다음 달 마케팅 전략 추천 |

- **목표 고객**: 마케팅에 적극적인 성장형 소상공인

## 8. 성장 전략 (단계별)

### Stage 1: 게릴라 론칭 (0~3개월)
**목표**: 1,000명 열성 사용자 확보

- **채널**: 인스타그램 소상공인 커뮤니티, 네이버 카페(소상공인마당), 자영업자 오픈채팅방
- **전술**: "우리 가게 SNS 만들어드립니다" 무료 체험 캠페인
- **목표 지역**: 서울 강남구·마포구·성동구 인스타 활발 상권 집중 공략
- **KPI**: 가입자 1,000명, 콘텐츠 생성 5,000건, 인터뷰 50건

### Stage 2: 바이럴 그로스 (4~6개월)
**목표**: 월 활성 사용자 10,000명

- **전술 1 — 소셜 증거**: 마케이트로 만든 콘텐츠에 자동 워터마크 → SNS 바이럴
- **전술 2 — 추천 보상**: 친구 추천 시 양측에 크레딧 100 지급
- **전술 3 — 유튜브 쇼츠**: "소상공인 SNS 5분 만들기" 콘텐츠로 유기 트래픽 확보
- **전술 4 — 지역 오프라인**: 동네 상공회의소·청년 창업 네트워크 데모 행사

### Stage 3: 채널 확장 (7~12개월)
**목표**: 월 사용자 50,000명, MRR 6,000만 원

- **B2G 파트너십**: 소상공인진흥공단·중기부 AI 바우처 공급기업 등록
- **프랜차이즈 B2B**: 가맹점 100개 이상 소형 프랜차이즈 본사 계약
- **언론 홍보**: "국내 최초 소상공인 특화 AI 마케팅" 포지셔닝으로 IT 매체 기고
- **SEO**: "소상공인 마케팅 무료", "카페 인스타 만들기" 등 롱테일 키워드 블로그 마케팅

### Stage 4: 플랫폼화 (2년차~)
**목표**: 월 사용자 200,000명, 흑자 전환

- **마케터 마켓플레이스**: 소상공인과 프리랜서 마케터 연결 수수료 수익
- **광고 연동**: 네이버 광고·카카오 광고 소재 자동 생성·집행 원스톱 서비스
- **데이터 비즈니스**: 업종별 마케팅 트렌드 리포트 B2B 판매

## 9. 리스크 및 대응

### 리스크 1: 캔바의 한국 시장 공격적 진입
- **위협도**: 높음 ★★★★☆
- **현황**: 2025년 캔바가 한국 TV 광고 시작, 적극적 한국 공략
- **대응**: 캔바가 커버하지 못하는 "소상공인 업종 특화 + 한국 문화 맥락" 집중 강화. 캔바는 글로벌 범용 플랫폼으로 소상공인 특화 기능 추가에 느림.

### 리스크 2: 미리캔버스의 AI 기능 고도화
- **위협도**: 높음 ★★★★☆
- **현황**: 미리캔버스는 이미 1,600만 가입자 + 2024년 780억 매출 + AI 기능 추가 중
- **대응**: 미리캔버스는 "디자인 툴"이고 마케이트는 "마케팅 결과 자동화 툴"로 포지셔닝 차별화. 템플릿 경쟁이 아닌 AI 자동화 결과 중심 경쟁.

### 리스크 3: AI 생성 콘텐츠 품질 한계
- **위협도**: 중간 ★★★☆☆
- **현황**: 현재 AI 이미지 생성 품질이 소상공인 실제 가게 분위기와 다를 수 있음
- **대응**: AI 생성 + 사용자 실제 사진 합성 기능 병행. 완전 AI 생성보다 "사용자 사진 기반 AI 보정" 방향 우선.

### 리스크 4: 낮은 유료 전환율
- **위협도**: 중간 ★★★☆☆
- **현황**: 소상공인은 디지털 서비스 유료 결제에 보수적
- **대응**: 첫 달 무료 체험 → 실제 매출 연결 사례 데이터로 전환 설득. "이 콘텐츠로 매출이 올랐어요" 후기 데이터 수집 및 노출 강화.

### 리스크 5: 정부 지원 의존도 과도화
- **위협도**: 낮음 ★★☆☆☆
- **현황**: B2G 채널 비중이 높으면 정책 변화에 취약
- **대응**: B2G는 30% 이하 비중 유지. 자체 유기 성장 채널(SEO, 바이럴) 병행.

## 10. 결론 및 권고사항

### 핵심 기회 요약

1. **시장 규모는 충분하다**: 한국 613만 소상공인 중 체계적 디지털 마케팅 실행 비율이 30%에 불과. 70%인 430만 소상공인이 마케팅 문제를 해결하지 못한 상태 — 이것이 마케이트의 시장이다.

2. **타이밍이 맞다**: 생성형 AI 마케팅 시장 CAGR 35.4% 성장 중. 정부의 소상공인 디지털 전환 지원 확대(최대 300만 원 지원). 대기업들이 이 시장에 집중하기 전 선점 가능한 골든타임.

3. **경쟁 공백이 존재한다**: "AI 자동화 × 소상공인 특화" 포지션을 차지한 경쟁사 없음. 캔바는 글로벌 범용, 미리캔버스·망고보드는 수동 디자인 툴, 뤼튼은 텍스트 AI — 모두 소상공인 마케팅 자동화에 초점이 없음.

### 핵심 실행 권고

| 순위 | 권고사항 | 이유 |
|------|----------|------|
| 1 | **MVP를 모바일 앱으로 먼저 출시** | 소상공인은 PC 아닌 스마트폰 위주. 웹보다 앱이 경쟁력 |
| 2 | **특정 업종 1~2개에 집중 시작** | "카페 전용 AI 마케팅" → "음식점 전용" 순으로 깊이 우선 |
| 3 | **소상공인 커뮤니티에서 풀뿌리 시작** | 네이버 카페, 오픈채팅방에서 100명 팬을 먼저 만들 것 |
| 4 | **정부 지원 사업 등록 선제 대응** | 소상공인진흥공단 AI 바우처 공급기업 등록으로 B2G 채널 조기 확보 |
| 5 | **"결과 증거" 수집을 핵심 KPI로** | "마케이트 쓰고 방문자 30% 증가" 류의 사례 10건이 최고의 마케팅 |

### 최종 메시지

> **"소상공인 사장님들은 마케팅을 하기 싫은 게 아니라, 할 시간이 없고 방법을 모르는 것이다."**

마케이트는 이 갭을 AI로 메운다. 복잡한 디자인 툴 배우거나 비싼 대행사 쓸 필요 없이, "오늘 메뉴 입력" → "5분 뒤 인스타 업로드"가 가능한 세상을 만든다. 소상공인의 생존율을 높이는 것이 마케이트의 진짜 미션이다.
